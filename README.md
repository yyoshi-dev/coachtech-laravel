# coachtech-laravel

## 概要

このリポジトリは、複数のLaravelプロジェクトをまとめた構成になっている。
各ディレクトリはそれぞれ独立したLaravel環境で、個別にセットアップして動作させる。

## ディレクトリ構成

- `advance-laravel`
  データ一覧・追加・更新・削除等が出来る学習用プロジェクト

- `contact-form`
  問い合わせフォーム実装プロジェクト

- `first-laravel`
  Laravelの基礎学習プロジェクト

- `third-laravel`
  Fortifyを使った認証周りの学習プロジェクト

- `todo`
  ToDoアプリ実装プロジェクト

## 各プロジェクトの使い方

各ディレクトリ内にREADMEを配置しているので、環境構築や実行方法はそれぞれのREADMEを参照する事。
